# Boardroom TV Website

Welcome to the Boardroom TV website project! This project is dedicated to showcasing content related to Boardroom TV, featuring a modern design with a blue, gold, and white color theme.

## Project Structure

The project is organized as follows:

```
boardroom-tv-website
├── src
│   ├── index.html          # Main HTML document for the website
│   ├── css
│   │   ├── style.css       # Main styles for the website
│   │   └── components.css   # Styles for specific UI components
│   ├── js
│   │   ├── main.js         # Main JavaScript functionality
│   │   └── video-cards.js  # Manages video cards for YouTube content
│   └── assets
│       └── logos
│           └── logo.svg    # Logo for Boardroom TV
├── package.json            # Configuration file for npm
└── README.md               # Documentation for the project
```

## Features

- **Responsive Design**: The website is designed to be responsive and user-friendly across various devices.
- **Video Cards**: Dynamic video cards that display YouTube content related to Boardroom TV.
- **Branding**: Incorporates the Boardroom TV logo and a cohesive color theme.

## Getting Started

To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd boardroom-tv-website
   ```

3. Open the `index.html` file in your web browser to view the website.
